Коли відбувається ререндер компонента:
1. Змінюється стан (завжди ререндериться)
2. Змінюється пропс  (завжди ререндериться)
3. відбувається ререндер батьківського компонента


React.memo(component) - мемоїзує компонент. Він не ререндериться, якщо пропси не змінюються (позбулись 3 пункта зверху).

useMemo(функція, [залежності]) - мемоїзує РЕЗУЛЬТАТ виконання функції. Функція повторно виклткається тільки при зміні залежностей

React.memo(IsFive, (prevProps, nextProps) => { 
    return false; - рендериться за замовчуванням
})

useCallback( () => {...}, [залежності]) - відміняє повторне створення функції
